# hittto 모바일 청첩장

정적 GitHub Pages에서 바로 작동하는 모바일 청첩장과 비개발자용 편집 작업창입니다.

## 주소

- 공개 청첩장: <https://hittto.github.io/>
- 편집 작업창: <https://hittto.github.io/editor.html>

## 편집 방법

1. 편집 작업창을 엽니다.
2. 왼쪽에서 글자, 날짜, 사진, 영상, 지도, 계좌번호, 색상을 수정합니다.
3. 오른쪽 모바일 미리보기로 결과를 확인합니다.
4. `GitHub에 게시`를 누릅니다.
5. 안내에 따라 `hittto/hittto.github.io` 저장소에 Contents 쓰기 권한이 있는 fine-grained token을 한 번 입력합니다.

토큰은 게시 요청에만 사용되고 편집 화면이나 저장소에 저장되지 않습니다. 수정 내용은 게시 전까지 현재 브라우저의 IndexedDB에 자동 저장됩니다.

## 주요 기능

- 세로형 모바일 청첩장 레이아웃
- 메인·중간·가족·영상 대표 사진 교체
- 여러 갤러리 사진 업로드, 순서 변경, 삭제, 확대/스와이프
- 유튜브 영상과 배경 음악 연결
- 자동 달력 생성
- OpenStreetMap 지도, 네이버 지도·카카오맵 연결
- 주소·계좌번호 복사와 모바일 공유
- JSON 백업/복원
- GitHub Git Data API를 통한 원클릭 게시

초기 사진은 교체 위치를 알기 위한 자체 제작 자리표시자입니다. 예시 사이트의 사진, 음원, 문구, 브랜드 자산은 포함하지 않았습니다.
