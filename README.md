# hittto 모바일 청첩장

클로드가 만든 모바일 청첩장 디자인과 기존 비개발자용 GUI 편집·GitHub 게시 기능을 결합한 정적 GitHub Pages 사이트입니다.

## 주소

- 공개 청첩장: <https://hittto.github.io/>
- 편집 작업창: <https://hittto.github.io/editor.html>

## 편집 방법

1. 편집 작업창을 엽니다.
2. 왼쪽에서 신랑·신부, 예식, 사진, 영상, 음악, 인사말, 연락처, 갤러리, 달력, 지도, 교통 안내, 계좌, 디자인과 공유 정보를 수정합니다.
3. 오른쪽 휴대폰 화면에서 클로드 디자인 미리보기를 확인합니다.
4. `GitHub에 게시`를 누릅니다.
5. `hittto/hittto.github.io` 저장소에 Contents 쓰기 권한이 있는 fine-grained token을 한 번 입력합니다.

토큰은 게시 요청에만 사용되고 편집 화면이나 저장소에 저장되지 않습니다. 수정 내용은 게시 전까지 현재 브라우저의 IndexedDB에 자동 저장됩니다.

현재 공개 내용은 `content.json`, 개인정보가 전혀 없는 초기화용 자리표시자는 `default-content.json`에 별도로 보관됩니다. 편집 화면에서 초기화를 눌러도 실제 이름·주소·계좌 정보가 기본값으로 재사용되지 않습니다.

편집기 페이지는 기본 비밀번호 `20181006`으로 잠겨 있습니다. 편집기의 **접속 비밀번호** 항목에서 편집기 비밀번호와 청첩장 방문 비밀번호를 각각 설정·변경·제거하거나 ON/OFF할 수 있습니다. 비밀번호 원문은 저장하지 않고 SHA-256 해시만 저장하며, 설정 변경은 **GitHub에 게시**한 뒤 공개 사이트에 적용됩니다. GitHub Pages는 정적 사이트이므로 이 기능은 일반 방문자를 위한 간단한 접근 잠금이며 서버 인증을 대체하지 않습니다.

## 공개 청첩장 기능

- 430px 모바일 중심 레이아웃과 스크롤 등장 모션
- 커버·서브 이미지와 포인트 색상·명조/고딕 글꼴
- 갤러리 더보기, 확대 보기, 키보드·모바일 스와이프
- 유튜브 또는 mp4 영상
- 자동 달력과 실시간 D-day 카운트다운
- 지도 배경은 네이버 지도 → 카카오맵 → 구글 지도 순으로 자동 선택하며, 네이버·카카오·구글 지도 연결 버튼 제공
- 연락처, 주소·계좌번호 복사, 계좌 아코디언
- 모바일 공유와 선택적 카카오 SDK 공유
- 배경 음악 재생 버튼과 자동 재생 시도

공개 화면은 `index.html` 한 파일에 클로드 디자인의 스타일과 동작을 포함하며, 편집 데이터는 `content.json`에서 불러옵니다.
